Écoutez votre radio préférée directement depuis WordPress avec Simple Radio Forty Two.

== Description ==

Simple Radio Forty Two est un plugin de lecteur de radio simple pour WordPress, inspiré par Radio 42. Ce plugin permet aux utilisateurs d'intégrer facilement un lecteur de radio directement dans leur site WordPress, offrant une expérience utilisateur fluide pour écouter des stations de radio en direct.

== Installation ==

1. Téléchargez le fichier ZIP de Simple Radio Forty Two sur votre ordinateur.
2. Connectez-vous à votre espace d'administration WordPress.
3. Allez dans **Extensions > Ajouter** et cliquez sur **Télécharger une extension**.
4. Chargez le fichier ZIP que vous avez téléchargé et cliquez sur **Installer maintenant**.
5. Activez le plugin depuis la liste des extensions pour commencer à le configurer.

== Configuration ==

1. Après l'activation, allez dans **Paramètres > Simple Radio Forty Two** pour configurer votre lecteur de radio.
2. Entrez l'URL de votre flux radio, choisissez le design et les options de personnalisation.
3. Optionnellement, vous pouvez personnaliser le style via le fichier de style CSS fourni avec le plugin.

== Utilisation ==

- Une fois configuré, vous pouvez ajouter le lecteur de radio dans n'importe quel contenu de votre site en utilisant le shortcode `[sr42_radio]` ou en plaçant le widget correspondant dans votre sidebar.

== FAQ ==

= Comment puis-je changer le design du lecteur ? =
Vous pouvez personnaliser le style du lecteur en modifiant le fichier CSS fourni ou en ajoutant des règles CSS dans votre thème.

= Le plugin fonctionnera-t-il avec n'importe quelle station de radio ? =
Oui, tant que vous fournissez une URL valide pour le flux de la station.

== Dépannage ==

- Assurez-vous que l'URL du flux est correcte et que le service de streaming est opérationnel.
- Vérifiez les journaux d'erreurs de WordPress pour tout message relatif au plugin.
